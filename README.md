# Regularbackup
A regular backup plugin based on [MCDReforged](https://github.com/Fallen-Breath/MCDReforged)

[Here](https://github.com/MCDReforged-Plugins/PluginCatalogue) is a MCDR plugin collection repository

```
MCDReforged/
├─ plugins/
│  ├─ Regularbackup.py
│  ├─ my_plugin1.py
│  └─ ...
│
├─ rb_temp/
│  ├─ Backup_file/
|  |  ├─ time1.zip
|  |  ├─ time2.zip
│  |  └─ ...
│  ├─ temp1/
│  └─ temp2/
│
└─ MCDReforged.py
```
